Simulated Enterprise Network Design & Expansion

Project Overview & Objective:
Collaborated with a four-member engineering team to analyze complex corporate requirements and design a robust network expansion for a global simulated business[cite: 28]. The project required outlining architectural expansion designs for corporate headquarters, branch offices, and regional data centers.

Technical Implementation & My Key Contributions:

  • London Headquarters Topology: Solely responsible for designing the high-level topology for the London HQ. This architecture featured dual core routers connected to an Internet Service Provider (ISP), dual core switches, a distribution layer switch, a wireless Access Point (AP), and a daisy-chained IP-phone-to-PC endpoint configuration.
  • Gateway Redundancy (HSRP): Configured and documented Hot Standby Router Protocol (HSRP) between core routers to guarantee seamless gateway redundancy and high network availability in case of hardware failure.

  • High-Level Design (HLD) Documentation: Authored "Section 4: Proposed Network Architecture" within the team's official High-Level Design document, clearly translating complex routing designs into business-ready explanations for stakeholders.
